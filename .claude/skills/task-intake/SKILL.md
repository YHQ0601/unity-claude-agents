# Task Intake

Evaluate before implementation:
1. What is the task type?
2. Is this necessary now?
3. What is the smallest useful version?
4. Which workstreams are required (architecture/code/assets/qa/memory)?
5. Should work start now?

Output:
## Task Type
## Necessity
## Smallest Useful Version
## Required Workstreams
## Should Start?
